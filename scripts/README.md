# scripts

Zona de scripts operativos durante migracion.

## Subcarpetas

- maintenance: tareas recurrentes y operativas.
- diagnostics: verificaciones y analisis.
- deprecated: scripts legacy sin uso activo confirmado.

## Politica

1. Mover, no eliminar, scripts legacy durante esta etapa.
2. Cada script debe indicar entrada, salida y riesgos.
3. Evitar rutas absolutas dependientes de una sola maquina.
